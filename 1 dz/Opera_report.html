Немного истории.

История берет начало в Норвегии. Два программиста Йон Стефенсон фон Течнер (слева) и Гейр Иварсёй (справа), создавая сайт для телефонной компании, где они работали, обнаружили, что существующий тогда браузер Mosaic не удовлетворяет их требованиям. Поэтому они принялись за создание собственного браузера.

Поставленные цели:

  • осуществление корректного просмотра страниц и документов;
  • быстрая скорость получения информации из Сети;
  • лёгкость и удобство работы;
  • небольшие размеры программы и инсталляционного пакета;
  • минимальные требования к ресурсам;
  • массовость использования.

Певая версия браузера выходит в 1994 году. В 1995 году версия была представлена на конференции по вопросам развития ПО, где получила множество положительных отзывов. После этого события создатели браузера решают основать свою фирму Opera Software AS.

C этого момента начинается активное развитие браузера.

О программе

Поддерживаемые стандарты


Начиная с девятой версии, Opera успешно проходит Acid2 — тест на соответствие самым современным технологиям веб-дизайна. Opera стала четвёртым браузером, прошедшим этот тест, и первым среди программ такого рода для Windows. Opera полностью проходит тест Acid3 начиная с 10-й версии.


Браузер Opera был одним из первых, поддерживающих CSS, ставшие к настоящему времени базовыми для веб-дизайна.


В настоящее время Opera поддерживает многие веб-стандарты:

  • CSS 3
  • HTML5
  • XHTML 1.1
  • XHTML Basic
  • XHTML Mobile Profile
  • XHTML+Voice
  • WAI-ARIA
  • WML 2.0
  • XSLT
  • XPath
  • XSL-F0
  • ECMAScript 3(JavaScript)
  • DOM 2
  • XMLHttpRRequest
  • HTTP 1.1
  • Юникод
  • SVG 1.1 Basic
  • SVG 1.1 Tiny
  • GIF89a
  • JPEG
  • APNG
  • PNG и в плане обработки альфа-канала

Пасхалки

Бета-версии Opera 9 содержали «пасхальное яйцо» — при запуске теста Acid2 жёлтый смайлик через некоторое время начинал отслеживать глазами курсор мышки. Если после этого на нём кликнуть, появлялось сообщение JavaScript «Because just passing is not enough ;)» — «Потому что простого прохождения недостаточно ;)».
